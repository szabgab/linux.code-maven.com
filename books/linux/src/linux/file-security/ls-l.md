# ls -l

```
drwxrwxr-x  8 vagrant vagrant 4096 Mar  7 20:08 Abc
-rwxrwxr-x  1 vagrant vagrant  476 Jun  6 15:39 a.sh
lrwxrwxrwx  1 vagrant vagrant    4 Jul  3 15:39 b -> a.sh
```

* 1 char  for type
* 3 chars for user (u)
* 3 chars for group members (g)
* 3 chars for others (o)


Type


```
- is file
d is directory
l is symbolic link
```

Rights


```
r - Read
w - Write
x - eXecute
```
